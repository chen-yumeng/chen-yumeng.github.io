CentOS7下安装Redis

一、安装redis

第一步：下载redis安装包

wget http://download.redis.io/releases/redis-4.0.6.tar.gz

第二步：解压压缩包

tar -zxvf redis-4.0.6.tar.gz

第三步：yum安装gcc依赖

yum install gcc

遇到选择,输入y即可

第四步：跳转到redis解压目录下

cd redis-4.0.6

第五步：编译安装

make MALLOC=libc　　

将/usr/local/redis-4.0.6/src目录下的文件加到/usr/local/bin目录

cd src && make install
